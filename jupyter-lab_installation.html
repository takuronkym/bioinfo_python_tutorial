Jupyter-Labのインストール: Mac編

Jupyter-LabはGoogle Colaboratoryと同様の機能をもつPythonの開発環境です。

🖥 インストール

下記のコードを実行してください。

brew install python3
pip3 install jupyterlab

実行後、terminalで下記のコマンドを実行してください。

jupyter-lab

ウェブブラウザが立ち上がり、下記のようなアプリケーションが開かれれば問題なくインストールができています。

確認できたら、Jupyter-labのツールバー > File > Shut Down でJupyter-labを終了し、ウェブブラウザのタブを閉じてください。
以上で準備終了です。
