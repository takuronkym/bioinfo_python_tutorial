📝 第三回

準備

第三回からはJupyterLabというPythonの開発環境（ソフト）を使います。MacとWindowsで導入方法が違いますので、それぞれのOSでのインストール方法を参照してください。

[Mac] Homebrewを介したインストール
 [Windows] Anacondaのインストール

教材

第三回以降は各自のパソコンでJupyterLabを開き、そこから配布されたJupyterNotebookを開いて進めます。

第三回の教材をダウンロード

    ダウンロードした圧縮ファイルをPCの任意の場所に解凍してください。
    [Mac]
    1. terminalを立ち上げ、解凍されたpython_tutorial_day3ディレクトリにcdしてください。
    2. 移動したらjupyter-labコマンドでJupyterLabを立ち上げてください。ウェブブラウザが起動しJupyterLabが開かれます。
    3. ファイルエクスプローラ（左側のペイン）からpython_tutorial_day3.ipynbというファイルをダブルクリックで開きます。
    4. 問題なく開けたら練習問題を解いてみてください。

    [Windows (Anaconda環境)]
    1. WindowsスタートメニューからAnaconda Navigatorを起動します。
    2. Anaconda NavigatorからJupyterLabを起動します。ウェブブラウザが起動しJupyterLabが開かれます。
    3. JupyterLabが起動したら、ファイルエクスプローラ（左側のペイン）で解凍されたpython_tutorial_day3ディレクトリを開きます。
    4. python_tutorial_day3.ipynbというファイルをダブルクリックで開きます。
    5. 問題なく開けたら練習問題を解いてみてください。

JupyterLabのインストールができなかった方は下記のGoogle Colaboratory上の教材を利用してください。
